tree is an acyclic graph
(درخت یه گراف بدون دوره)

tree (درخت) :
 each node is a tree(even by it self)
 (هر گره یه درخته (حتی به تنهایی))

node (گره) :
 an element of a tree
 (یه گره از درخت)

root (ریشه) :
 a node with no parent
 (گره ای که پدر نداره)

leaf (برگ) :
 a node with no children
 (گره ای که بچه نداره)

sibling (همزاد) :
 nodes with the same parent
 (گره هایی با یک پدر)

forest (جنگل) :
 a collection of trees
 (مجموعه ای از درخت ها)


ancestors (اجداد) :
 all nodes on the path from a node to the root
 (همه گره های موجود تو یه مسیر از گره تا ریشه)

sub tree (زیردرخت) :
 a node and all nodes below it (a node with all of its descendants)
 (یه گره و همه گره های زیرش (یه گره با همه فرزنداش))

descendants (فرزندان) :
 all nodes in a tree or subtree with a given root
 (همه گره های درخت یا یه زیردرخت با ریشه مشخص)

internal node (گره داخلی) :
 all nodes except leaf nodes
 (همه گره ها بجز برگ ها)


tree size (اندازه درخت) :
 number of nodes
 تعداد گره ها

tree depth (عمقِ درخت) :
 root is at depth(level) 0 and each lower node is one level deeper
 (ریشه تو عمق 0عه و هر گره زیرش یه سطح عمیق تره)

node depth (عمق گره) :
 given node is at depth(level) 0 and each lower node is one level deeper
 (گره مشخص شده تو عمق 0عه و هر گره زیرش یه سطح عمیق تره)

tree height (ارتفاع درخت) :
 maximum path length from the root to a leaf (maximum number of edges from root node to a leaf node)
 (بیشترین طول مسیر(فاصله) از ریشه تا برگ)

node hight (ارتفاع گره) :
 maximum path length from a node to a leaf
 (بیشترین طول مسیر(فاصله) از گره مشخص شده تا برگ)

JUST A TIP :
 when we go down from the root, depth increases
 and height decreases because we get closer to the leaves
 ( وقتی از ریشه پایین تر میریم عمق بیشتر
   و ارتفاع کمتر میشه چون به برگ ها نزدیک تر میشیم )


k-array tree :
 each node can have at most k children
 (هر گره میتونه k تا بجه داشته باشه)

full k-array tree :
 each node has k children or no children
 (هر گره یا k تا بچه داره یا هیچ بچه ای نداره)

balanced tree (درخت متعادل) :
 leaf depths difference at most is one
 (اختلاف عمق برگ ها حداکثر یک‌عه)

completely balanced tree (درخت کاملا متعادل) :
 all leaf nodes have the same depth
 (عمق همه برگ ها یکسانه)

complete k-array tree :
 tree filled from left to right 
 (nodes are inserts in level-first order from left to right, it is balanced)
 (درختی که از سمت چپ به راست پر شده)
 ((گره ها به صورت اول سطح از چپ به راست وارد میشن، متعادل هم هست))

ordered tree (درخت مرتب) :
 order of children is important
 (ترتیب فرزند ها مهمه)

labeled tree (درخت برچسب دار) :
 each node has a label
 (هر گره یه برچسب داره)

binary tree (درخت دودویی) :
 a k-array tree where k equals 2
 each node has data, a left child, and a right child at least
 ( یه درخت که هر گره حداکثر دو فرزند داره
   هر گره حداقل دارای داده، فرزند چپ و فرزند راست هست )
 for more : go to ../tree/binary_tree/
